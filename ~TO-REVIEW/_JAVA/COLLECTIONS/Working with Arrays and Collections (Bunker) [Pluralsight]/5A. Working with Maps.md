# HashMap

```java
Map<String, String> map = new HashMap<>();
map.put("one", "ONE");
String get = map.get("two");
```

-   `put()` to add an entry
-   `get()` to get the value

# TreeMap

-   `TreeMap` automaticly sort the values at each insertion
-   it stores the value separatly in a _red-black tree structure_
